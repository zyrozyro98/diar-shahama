# معرض ديار الشهامة - Diar Al Shahama

تطبيق ويب متكامل لإدارة معرض السيارات (ديار الشهامة). يتميز بتصميم عصري وفاخر مع لوحة تحكم متكاملة.

## المميزات
- **الصفحة الرئيسية**: عرض جذاب مع قسم للإحصائيات وأحدث السيارات.
- **عرض السيارات**: قائمة تفاعلية مع إمكانية التصفية حسب النوع (سيدان، دفع رباعي، رياضية).
- **توثيق المبيعات**: قسم لعرض فيديوهات تسليم السيارات للعملاء (مبيعات سابقة).
- **نظام الحجز**: نموذج سهل للعملاء لحجز موعد معاينة أو تجربة قيادة.
- **لوحة التحكم (Admin)**: 
  - إدارة السيارات (إضافة، حذف، تعديل).
  - إدارة توثيق المبيعات (نشر فيديوهات جديدة).
  - مراجعة الحجوزات الواردة.
  - حماية الدخول للمسؤولين فقط.

## التقنيات المستخدمة
- **HTML5 & Vanilla CSS**: لتصميم فريد وسريع.
- **JavaScript (ES6+)**: للتعامل مع المنطق والبيانات.
- **Firebase**: قاعدة البيانات (Firestore) والمصادقة (Auth).
- **Vite**: كأداة بناء وتطوير.

## طريقة التشغيل
1. تأكد من تثبيت Node.js.
2. قم بتشغيل الأمر التالي في المجلد الرئيسي:
   ```bash
   npm install
   ```
3. لتشغيل الموقع في بيئة التطوير:
   ```bash
   npm run dev
   ```
4. لفتح لوحة التحكم:
   - الرابط: `قسم المسؤول` في القائمة العلوية.
   - البريد: `admin@diar.com`
   - كلمة المرور: `123456`

## هيكلة المشروع
- `index.html`: الملف الرئيسي.
- `src/style.css`: التصميم الفاخر (Dark Mode & Gold accents).
- `src/main.js`: المنطق التفاعلي.
- `src/firebase-config.js`: إعدادات الربط مع قاعدة البيانات.
